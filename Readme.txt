**Project Title:**
**Smart Parking Management System Using Data Structures**

**Project Description:**

The Smart Parking Management System is a standalone desktop application designed to make parking management easier and more efficient in places such as shopping malls, hospitals, educational institutions, and office complexes. The system helps users find, reserve, and manage parking spaces while providing administrators with an organized way to monitor parking activities.

**Main Functionalities / Modules:**

* **Parking Slot Allocation Module**

  * Automatically assigns an available parking slot to a vehicle.
  * Updates slot status when a vehicle enters or exits.

* **Parking Reservation Module**

  * Allows users to reserve parking slots in advance.
  * Prevents double booking of reserved slots.

* **Vehicle Search and Management Module**

  * Searches for parked vehicles using registration numbers.
  * Displays parking details and slot information quickly.

* **Waiting Queue Management Module**

  * Maintains a queue of vehicles when all parking slots are occupied.
  * Automatically allocates a slot when one becomes available.

* **Parking Status and Report Module**

  * Displays available and occupied parking slots.
  * Generates basic parking statistics and reports.

**Data Structures Used:**

* **Arrays** for parking slot management.
* **Queue** for handling waiting vehicles.
* **Hash Map** for fast vehicle search and retrieval.
* **Linked List** for maintaining parking records and history.

This project combines practical real-world utility with the implementation of important Data Structures, making it an effective and meaningful application for learning and demonstrating DSA concepts.
